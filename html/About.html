はじめまして。Webデザイナーを目指して日々制作に取り組んでいます。 デザインカンプの作成からHTML/CSSによるコーディングまで、一貫して対応することができます。 見る人にとってわかりやすく、使いやすいシンプルなデザインを心がけて制作しています。

Skill Set

Photoshop

ロゴの作成、写真のレタッチ、デザインカンプの作成、バナー制作などに使用しています。ビジュアルの印象や訴求力を意識しながら、細部まで丁寧に仕上げることを心がけています。

Illustrator

LINEスタンプやロゴの作成など、ベクターイラストを用いた制作に使用しています。Photoshopでは表現が難しいグラデーションや細かな調整が必要なデザインも、Illustratorで対応しています。印象に残る、視認性の高いビジュアル制作を心がけています。

Dreamweaver

Web制作の学習初期に、HTML/CSSのコーディングをDreamweaverで行っていました。現在では、より柔軟な編集ができるVisual Studio Codeに移行し、効率的なコーディング環境で制作を行っています。

Figma

Webサイト制作において、フレームワークの作成からモックアップ、デザインカンプの制作まで一貫して行っています。構造を意識した設計と、視認性の高いシンプルなデザインを心がけています。

HTML

誰が見てもわかりやすいコーディングを心がけ、クラス名やタグの構造に一貫性を持たせるよう意識しています。可読性と保守性を重視したマークアップを大切にしています。

CSS

デザインを忠実に再現することはもちろん、再利用しやすいスタイル設計やレスポンシブ対応を意識しています。さらに、アニメーションの実装やスムーススクロールなどの動きの演出も可能で、ユーザーにとって心地よい操作感を大切にしています。

Jquery

動的なコンテンツの操作やユーザーインタラクションの実装が可能です。基本的なイベント処理やアニメーション、フォームの動作改善などに対応しています。
